# AntTestTool
## A test tool for Langton's ant extensions. Can be used in combination with the Nist-statistical-test-suite.

### Prerequisites
#### Mandatory
Scala library container version 2.12.3
Java 1.8

#### Optional
Scala IDE
SBT
https://github.com/terrillmoore/NIST-Statistical-Test-Suite
